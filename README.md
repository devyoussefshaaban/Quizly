## Overview about Quizly

Quizly is a web application that allows users to create and take quizzes.
It is built using the latest web technologies and is designed to be easy to use and navigate.

## Main Tech Stack

### Frontend
- React
- Redux
- TypeScript
- Material UI
- Sass

### Backend
- Node.js
- Express
- MongoDB
- BcryptJS
- JsonWebToken

## Main Dev Dependencies

- nodemon
- concurrently
